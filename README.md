# Noob_Xiaoyu 个人作品集网站

这是一个基于 Vue 3 + Element Plus 构建的个人作品集网站，展示了我开发的各种开源项目和工具。

## 🚀 项目特色

- **现代化 UI 设计**：采用深色/浅色主题切换，响应式布局适配各种设备
- **项目展示**：清晰展示每个项目的功能、技术栈和在线预览
- **下载中心**：提供部分项目的可直接下载版本
- **技术博客**：分享开发经验和教程（规划中）

## 📦 包含的项目

### 🎮 SteamAccountManager
现代化的 Steam 账号管理桌面应用，支持账号分类、批量操作、自动登录、深色/浅色主题等功能。

**技术栈**：C++, WebView2, Windows API, Vue 3, Vite

### 🎨 Imgui-menu
基于 ImGui 的现代化悬浮菜单框架，集成了辉光、鼠标拖尾等丰富视觉特效和计时器等实用工具。

**技术栈**：C++, ImGui, OpenGL

### 🤖 launcher for napcat&yunzai
在 Windows 电脑上的一键启动和运行 QQ 机器人的“傻瓜式”工具。

**技术栈**：C++

### 📥 yt-dlp-gui
为 yt-dlp 开发的图形用户界面 (GUI) 工具，让不熟悉命令行的用户也能轻松使用其强大的视频下载功能。

**技术栈**：C++, ImGui

### 🗂️ Minecraft 投影文件分类
用于将 Minecraft Litematica 模组的投影文件按文件夹进行自动化分类整理的小工具。

**技术栈**：C++

### 🔄 ImageBinaryConverter
可以在图片和二进制文件之间进行双向转换的工具，用于文件伪装、隐藏和绕过传输限制。

**技术栈**：C++, ImGui

### 🖥️ ImGui-Main
基于 Dear ImGui 的 C++ 图形界面应用程序框架，使用 OpenGL 作为渲染后端，提供完整的 UI 控件库和渲染功能。

**技术栈**：C++, ImGui, OpenGL

## 🛠️ 本地开发

### 环境要求
- Node.js 20+ 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

构建后的文件将生成在 `dist` 目录中。

### 运行构建脚本
Windows 系统可以直接运行 `Run Build.bat` 进行一键构建。
